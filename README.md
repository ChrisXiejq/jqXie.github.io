# AI Oral Learning Blog (GitHub Pages)

This repository hosts the static GitHub Pages site for an academic blog on conversational AI for oral English learning.

## Contents

- `index.html`: single-file webpage.
- `figures/`: image assets used by the webpage.

## Update Workflow

When the source article is updated:

1. Regenerate/update webpage content in the source project.
2. Copy the latest `index.html` into this repository root.
3. Copy/update the `figures/` directory.
4. Commit and push changes to publish via GitHub Pages.

## Notes

- Keep links and asset paths relative.
- Avoid adding personal identifiable information to repository metadata.
